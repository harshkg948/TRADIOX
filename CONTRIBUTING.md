# Contributing to TRADIOX 🚀

We welcome contributions!

## How to contribute:
1. Fork the repository
2. Clone your fork
3. Create a new branch (feature-name)
4. Make your changes
5. Commit and push
6. Create a Pull Request

## Rules:
- Keep code clean
- No API keys
- Follow project structure

Let’s build India’s first open-source trading AI 🔥
